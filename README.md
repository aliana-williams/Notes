# Lecture-to-Notes Summarizer

## Overview
This Python application converts lecture PDFs into summarized study notes. Users can upload lecture documents, choose a summary length through a graphical user interface (GUI).

## Features
- Uploads PDF lecture files
- Extracts text from PDFs
- Uses AI to generate summarized notes
- Multiple summary length options
- Simple graphical user interface using Tkinter

## Technologies Used
- Python
- Tkinter
- OpenAI API

## How It Works
1. User uploads a lecture PDF.
2. The program extracts text from the document.
3. The text is split into smaller chunks for processing.
4. The OpenAI API summarizes the content into organized notes.
5. The summarized notes are displayed inside the application.
